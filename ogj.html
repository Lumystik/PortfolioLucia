OGJ Project Overview

Project Introduction: The OGJ (Observatory of Gender and Youth) website is a platform that promotes gender equality and addresses gender-based violence among youth. It's designed to inform, support, and engage young users and the public.

Team and Guidance

Team Members: Lucía Medina, Berenice Castro, Frida Castro

Guided by Professor Elizabeth Hernández Rosas.

Solution Strategies

Our project strategies included analyzing client and user needs, defining user profiles, setting front and back-end objectives, and creating a prototype.

User and Client Needs

Client Needs

  • Friendly, non-institutional tone
  • Inclusive and engaging
  • Educational and simple interface

User Needs

  • Realistic and practical resources
  • Accessible information for youth
  • Supportive resources for all users

User Profiles

We identified sample profiles to guide design:

  • Mariana: 20, facing relationship violence, hesitant to seek help.
  • Rebeca: 42, researcher focused on gender violence in Mexico.
  • Víctor: 22, supporting friends affected by gender violence.

Development Objectives

Front-End

  • Create a site that raises awareness on gender violence
  • Engage users with interactive content
  • Present information clearly without overwhelming users

Back-End

  • Simple content management system
  • Intuitive workspace for administrators
  • Streamlined, easy-to-manage features

Prototype and Design Process

The team used mood boards, wireframes, and a business model canvas to map user interactions and define essential sections.

We applied Design Thinking, card sorting, and wireframing for consistent and effective page structure.

Content and Interaction

  • Learning Sections: FAQs, articles, statistics on gender equality
  • Forums: Safe space for discussions
  • Victim Support: Maps and directories for help centers

Final Features

  • Simple, user-friendly navigation
  • Minimal text and clutter-free layout
  • Easy-to-use back-end management for responsiveness
